1. Responsive Size

  • Mobile → ছোট font size (text-3xl)
  • Tablet → medium (sm:text-4xl)
  • Desktop → বড় (md:text-5xl)

    • Font Weight

    Light, normal, semi-bold, bold?
    Tailwind → font-semibold / font-bold

    Text Color

    White? Black? Brand color?
    Tailwind → text-white / text-black / text-[#B68C5A]

    Text Alignment

    Centered, left, right?
    Tailwind → text-center / text-left / text-right

    Max Width

    Mobile full width, Desktop limited width
    Tailwind → max-w-xl / md:max-w-[690px]

    Line Height

    Readability → line height কত?
    Tailwind → leading-snug / leading-[1.5]

    Margin / Padding

    Section spacing, top-bottom margin, spacing with next element
    Tailwind → mt-4 / mb-8 / px-4

    ✅ Summary Thinking Flow
    Responsive → size, layout
    Font → weight
    Color → contrast, readability
    Alignment → center / left / right
    Max width → mobile vs desktop
    Line height → readability
    Margin / Padding → spacing
